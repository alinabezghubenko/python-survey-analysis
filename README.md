# Python Survey Analysis

## Project Overview
Exploratory Data Analysis (EDA) of a developer survey dataset using Python.  
The project focuses on identifying trends in experience, salary, remote work, and Python adoption.

---

## Objectives
- Clean and preprocess raw survey data
- Analyze work experience distribution
- Explore remote work trends
- Investigate salary patterns
- Identify Python adoption across different groups

---

## Tools Used
- Python
- Pandas
- Matplotlib

---

## Key Insights
- Remote work shows noticeable impact on salary distribution
- Experience level strongly correlates with compensation
- Python remains popular among mid-level developers
